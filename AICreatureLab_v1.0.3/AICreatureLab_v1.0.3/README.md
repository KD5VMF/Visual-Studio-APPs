# AI Creature Lab

Version: 1.0.3

Windows Forms C# evolutionary creature simulator tuned for a higher-workload workstation profile.

## Highlights
- live 2D evolution sim
- neural-net creatures
- champion save/load
- parallel update path using up to the machine's logical processor count
- heavier default population/resource targets for better CPU utilization
- on-screen version + tick timing display

## Open
Open `AICreatureLab.sln` in Visual Studio and run on Windows.
