# Contributing

Thanks for helping make PFLT14 better.

Good contributions include:

- Cleaner dashboard layout improvements.
- Better SNMP interface detection.
- Safer unit scaling and counter handling.
- Clearer setup documentation.
- Bug fixes that keep the app simple and useful for home-lab users.

Please do not commit private router addresses, SNMP community strings, screenshots with secrets, logs with private network details, or files from `Documents\PFLT14` unless they have been reviewed and sanitized.
